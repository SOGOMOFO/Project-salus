# Project Salus Constitution v1.0

## Mission

Project Salus exists to increase human judgment, resilience, freedom, execution, and legacy through evidence-based intelligence and disciplined action.

## Mission Zero

Every subsystem must improve decision quality.

If a feature does not improve intelligence, judgment, execution, learning, security, or legacy, it is lower priority.

## Core Loop

Reality → Evidence → Judgment → Decision → Execution → Outcome → Review → Learning → Memory → Better Future Decisions

## Founder Principle

Project Salus should make its users progressively wiser, more capable, and less dependent on AI over time.

## Human Accountability

AI may advise, analyze, summarize, forecast, and challenge assumptions.

The human remains accountable for final decisions.
