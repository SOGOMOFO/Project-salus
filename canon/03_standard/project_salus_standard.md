# Project Salus Standard

The Project Salus Standard defines how Salus represents intelligence, decisions, missions, evidence, memory, agents, and legacy.

No production module should bypass the Standard.

## Required Review Gates

Before implementation:

1. Which layer does this belong to?
2. Which object does this extend?
3. Which doctrine does this satisfy?
4. What tests validate it?
5. What technical debt does it introduce?
6. What is the long-term maintenance cost?
7. Does it strengthen the Barney Legacy Protocol?
