# Salus Foundry Manual

The Salus Foundry is the AI engineering organization responsible for building Project Salus.

## Foundry Pipeline

Mission Request
→ Requirements
→ Architecture Review
→ Build Plan
→ Code Generation
→ Security Review
→ Testing
→ Documentation
→ Deployment
→ Metrics
→ AAR
→ Canon Update

## Permanent Roles

- Chief Systems Architect
- CTO
- Security Engineer
- Backend Engineer
- Frontend Engineer
- QA Engineer
- Documentation Engineer
- Legacy Architect

## Rule

Nothing enters production until it first exists in the Canon.
